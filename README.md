# 🔥 ポケトレ

5〜10分の脂肪燃焼サーキットを、タイマーに従うだけで実行できるパーソナルトレーナーアプリ。
キャラクター「メラ」がレベルに応じて5段階に進化し、連続記録(ストリーク)とXPで楽しく継続できます。

## 機能

- ⏱ **サーキットタイマー**:30秒ワーク+15秒休憩。ビープ音でカウントダウン、休憩中に次の種目を予告
- 🏃 **3コース**:サクッと5分(6種目)/ しっかり8分(10種目)/ 本気の10分(12種目)— すべて器具なしの自重トレ
- 🔄 **日替わりメニュー**:12種目からその日のセットが自動で入れ替わる
- 🎥 **3D動作デモ**:全12種目の動きをThree.js製の3Dフィギュアがループ実演。ホームのメニュー名タップ、またはワークアウト中の「動きを3Dで見る」から表示
- 🔥 **メラの育成**:たまご → ひだね → メラ → ブレイズ → フェニックスの5段階進化
- ⚡ **ストリーク&レベル**:連続日数でXPボーナス増加。「かけだし」から「レジェンド」までのランク制
- 💦 **消費カロリー**:体重設定に基づきワークアウトごとの推定消費カロリーを計算・記録(8METs換算)
- 🍽 **食事記録**:定番食品60種超のクイック登録。入力欄に品名を打つと目安カロリーを自動補完(ひらがな・カタカナの表記ゆれ対応)。目標カロリー+運動消費分に対する残量を表示
- ⚖️ **体重記録**:毎日の体重を手動入力し、推移を折れ線グラフで確認
- 📅 **デイリー記録**:日付をさかのぼって過去のアクティビティ・食事・体重を閲覧&編集
- 📊 **記録画面**:直近7日間のグラフ、累計回数・時間・消費カロリー、ワークアウト履歴
- 📱 **iPhone対応PWA**:ホーム画面に追加してフルスクリーンの独立アプリとして動作。記録はlocalStorageに保存され、ワークアウト中は画面スリープを防止(Wake Lock対応端末)

```
└── README.md
```

## デプロイ(GitHub Pages)

1. このリポジトリの中身をそのままプッシュ
2. リポジトリの **Settings → Pages** で `main` ブランチの `/ (root)` を公開対象に設定
3. 表示されたURL(`https://<ユーザー名>.github.io/<リポジトリ名>/`)にアクセス

## iPhoneへのインストール

1. iPhoneの **Safari** で上記URLを開く
2. 共有ボタン(□↑)→ **「ホーム画面に追加」**
3. ホーム画面のアイコンから起動すると、フルスクリーンのアプリとして動作します

## 注意事項

- 記録はブラウザのlocalStorageに保存されます。Safariのデータ消去や「履歴とWebサイトデータを消去」を行うと記録もリセットされます
- 運動は無理のない範囲で行い、体調に不安がある場合は医師に相談してください
